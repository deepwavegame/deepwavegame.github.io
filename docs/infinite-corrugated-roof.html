Infinite Corrugated Roof Documentation

Complete documentation for Infinite Corrugated Roof — a live, procedural corrugated-metal roof generator for Unity with automatic LODs and FBX baking.

📄️ Overview

Infinite Corrugated Roof generates production-ready corrugated metal roofs in Unity from a single component — live panels, curve wave profiles, weathering, spline deformation, 3-level LOD and FBX baking.

📄️ Introduction

Infinite Corrugated Roof (ICR) is a live, procedural corrugated-metal roof generator for Unity built on a single component with instant Inspector rebuilds and automatic LODs.

📄️ Getting Started

Create your first procedural corrugated roof in Unity — add the Infinite Corrugated Roof component, assign a material, and size the panel grid live in the Inspector.

📄️ Editor Guide

Every field on the Infinite Corrugated Roof Inspector — panel layout, wave configuration, mesh, modifiers, texture atlas mapping and randomization.

📄️ Advanced Features

Spline deformation, angled edge cuts and one-click FBX baking for Infinite Corrugated Roof — optional workflows that ship static, deterministic geometry.

📄️ LOD & Performance

How Infinite Corrugated Roof builds three LOD meshes, its screen-relative transition heights, and why live editing produces no garbage-collector pressure.

📄️ API Reference

Script the Infinite Corrugated Roof component — every parameter is a public field; call Rebuild() to apply changes, plus baked-mesh helpers.
